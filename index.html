AddressBook

A lightweight, file-based contact management application written in C, designed with reliability, simplicity, and clarity in mind.

Source Code Watch Demo

Why AddressBook?

Persistent Storage

Contacts are stored using file handling, ensuring data is retained across executions.

Menu Driven Design

Simple terminal-based interface with clear options for user interaction.

Robust Validation

Prevents invalid inputs and ensures reliable contact operations.

Key Features

  • Add, view, search, edit, and delete contacts
  • Structured data management using C structures
  • Modular source code for maintainability
  • Efficient file I/O operations
  • Designed for Linux environments

Technology Stack

C Programming File Handling Structures Linux Strings Pointers
