Address Book Application

C Programming | File Handling | Modular Design

Project Overview

This project is a menu-driven Address Book application written in C, designed to manage contact information efficiently using file-based storage. It focuses on applying core C programming concepts to a real-world problem.

What I Built

  • Implemented add, search, edit, delete, and display contact operations
  • Stored contact data persistently using file handling
  • Validated user inputs to prevent invalid data entry
  • Designed the application using modular programming

How I Built It

  • Used structures to organize contact details
  • Applied file I/O functions for persistent storage
  • Separated logic into multiple source files
  • Followed a clean, readable, and maintainable coding style

Impact & Learning

This project strengthened my understanding of C fundamentals such as structures, pointers, file handling, and control flow. It improved my problem-solving approach and helped me write cleaner and more reliable code.
